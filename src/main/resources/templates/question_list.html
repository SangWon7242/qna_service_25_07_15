속성 설명 값
paging.isEmpty 페이지 존재 여부를 의미한다(게시물이 있으면 false, 없으면 true).
paging.totalPages 전체 페이지 수를 의미한다.
paging.totalElements 전체 게시물 수를 의미한다.
paging.size 한 페이지에 보여지는 게시물 수를 의미한다.
paging.number 현재 페이지 번호를 의미한다.
paging.numberOfElements 현재 페이지에 보여지는 게시물 수를 의미한다.
paging.first 첫 페이지 여부를 의미한다.
paging.last 마지막 페이지 여부를 의미한다.
paging.hasNext 다음 페이지가 존재하는지 여부를 의미한다.
paging.hasPrevious 이전 페이지가 존재하는지 여부를 의미한다.
paging.sort 현재 페이지에 적용된 정렬 기준을 의미한다.
paging.pageable 현재 페이지에 적용된 페이징 정보(페이지 번호, 크기 등)를 의미한다.
